# Elzero – Responsive Multilingual Website Template

A modern, fully responsive **business website template** built with **HTML5, CSS3, JavaScript, and Bootstrap 5**, featuring **English/Arabic multilingual support** with automatic **RTL/LTR switching**.

## Project Overview

This project demonstrates how to build a **production-quality static website** using vanilla web technologies while maintaining:

- Clean layout structure
- Responsive design across devices
- Smooth UI interactions
- Multilingual support with RTL handling
- Modular, maintainable front-end code

The focus is on **frontend fundamentals and UI behavior**, not backend logic.

---

## Core Features

### Responsive & Layout
- Mobile-first responsive design
- Bootstrap 5 grid system
- Adaptive layouts for desktop, tablet, and mobile

### Multilingual Support
- English (LTR) and Arabic (RTL)
- Instant language switching
- Persistent language preference
- Separate RTL stylesheet for Arabic

### UI & Interactions
- Mega menu navigation
- Scroll-triggered animations
- Animated counters and progress bars
- Event countdown timer
- Scroll-to-top button
- Hover effects and transitions

- Events with live countdown

## Architecture & Structure

The project is organized for clarity and scalability:

- **HTML** for semantic structure
- **CSS** split into LTR and RTL stylesheets
- **JavaScript** for interaction and animation logic
- **JSON translation files** for internationalization

This separation makes the template easy to customize or extend.

---

## Technology Stack

- **HTML5** – Semantic markup
- **CSS3** – Flexbox, Grid, animations
- **JavaScript (ES6)** – DOM logic, animations, counters
- **Bootstrap 5** – Layout and components
- **Font Awesome** – Icons
- **i18next** – Language translation handling

---

## Engineering Focus

This project emphasizes:
- Frontend structure and readability
- Responsive UI design
- RTL/LTR layout handling
- Clean JavaScript interaction logic
- Reusable and customizable template design

It is intended as a **frontend showcase template**, not a backend-driven application.
